# MotoCalc (Windows)

Повноцінний додаток з одним вікном для роботи з файлами:
- 6055.xls (джерело)
- 6055_MOTO_template.xls (акт)
- DOGOVIR_6055_template.doc (договір)
- vidatkova.xls (видаткова)

## Що робить програма

- Дає одне велике вікно з полями у стилі головного акта 6055.
- Підсвічує порожні або підозрілі поля.
- Показує попередження, якщо C39/C42/C43 не збігаються.
- Автоматично перевіряє бухгалтерську логіку: `ціна без ПДВ + ПДВ = ціна з ПДВ`.
- Перевіряє формати дати, ІПН / коду, VIN / рами та номера двигуна.
- Має окремі кнопки:
  - Заповнити з 6055
  - Очистити форму
  - Акт
  - Договір
  - Видаткова
  - Генерувати
- Дає вибір, чи відкривати готовий файл через зовнішній редактор/софт.
- Результати пише в: %USERPROFILE%\Documents\MotoCalc\out
- Кнопка `Генерувати` створює окрему папку покупця за скороченим ПІБ і кладе туди всі три документи.

## Запуск на Python (fallback)

```powershell
run_app.bat
```

Це запасний режим, якщо антивірус блокує exe.

## Збірка EXE

```powershell
build_exe.bat
```

Після збірки запускати:

```powershell
dist\MotoCalc\MotoCalc.exe
```

## Порада щодо антивірусу

- Менше false positive зазвичай дає режим onedir (вже використано), а не onefile.
- Не використовувати упаковку/обфускацію.
- У збірці вимкнено UPX (`--noupx`), щоб зменшити спрацювання антивірусу.
- Збирати на чистій Windows 10/11 VM.
- Якщо є корпоративний AV на Windows Server 2019, додати папку релізу у allowlist.
- Для production бажано підписувати exe code-sign сертифікатом.

## Як працює інтерфейс

1. Головна вкладка `Дані` виглядає як велика форма акта з тематичними блоками.
2. Кнопка `Заповнити з 6055` перечитує дані з файлу [6055.xls](6055.xls).
3. Кнопка `Очистити форму` скидає поля для нового введення.
4. Кнопки `Акт`, `Договір`, `Видаткова` відкривають окреме вікно чорновика.
5. Чорновики показуються як документний макет, а не просто як текстовий preview.
6. У вікні чорновика кнопка `Зберегти` створює фінальний файл у потрібному форматі.
7. Кнопка `Генерувати` створює акт, договір і видаткову за один прохід.
8. Для повної генерації створюється нова папка кейсу на ім'я скороченого ПІБ покупця.
